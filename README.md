# Delivery Website

This is a starter project scaffold for a delivery tracking website.

Structure:
- backend: Express API, models, routes
- frontend: React components and public assets
- utils: helper utilities

Next steps:
- Initialize git
- Install dependencies
- Implement models and real endpoints

CI Status
---------

Add this repository to GitHub and push the branch to enable the workflow. After pushing, the workflow will run on every push and pull request to `main`.
